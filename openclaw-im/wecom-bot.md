# 企业微信长连接机器人接入指南

> 适用版本：OpenClaw >= 2026.3.13 · 企业微信最新版客户端

---

## 一、这是什么？

企业微信支持通过「长连接智能机器人」接入 OpenClaw。接入后，你可以在企业微信的私聊和群聊中与你的 OpenClaw Agent 直接对话。

**与微信 ClawBot 的区别：**

| 特性 | 微信 ClawBot | 企微长连接机器人 |
|------|-------------|----------------|
| 私聊 | ✅ | ✅ |
| 群聊 | ❌ 不支持 | ✅ @机器人触发 |
| 主动推送 | ❌ 有限 | ✅ 支持 |
| MCP 能力 | ❌ | ✅ 文档、日程、会议、智能表格 |
| 需要公网 IP | ❌ | ❌（长连接模式） |
| 开放状态 | 灰度放量中 | 全量开放 |

**核心优势：** 长连接模式不需要域名、不需要公网 IP、不需要配置 IP 白名单，部署门槛极低。

---

## 二、前提条件

| 项目 | 要求 |
|------|------|
| OpenClaw | 已安装并正常运行 |
| 企业微信账号 | 需要管理员权限（个人可免费注册企业） |
| 企业微信客户端 | 手机端安装最新版 |

> **没有企业微信账号？** 前往 https://work.weixin.qq.com 注册，个人用户可以创建一个新企业，自动成为管理员。

## 三、安装步骤

### Step 1：在企业微信后台创建智能机器人

1. 登录企业微信管理后台：https://work.weixin.qq.com
2. 左侧导航栏 → **管理工具** → **智能机器人**
3. 点击 **「创建机器人」** → **「手动创建」**
4. 在创建页面底部找到小字：*"如需使用自有系统获取成员与机器人的聊天并输出回复，可切换至 API 模式创建"*，点击 **「API 模式创建」**
5. 连接方式选择 **「使用长连接」**

> **重要：** 必须选择「长连接」而非「URL 回调」。长连接不需要公网 IP 和域名。

6. 页面会自动生成两个值：
   - **Bot ID**
   - **Secret**

   **立即复制保存这两个值！** 丢了只能重新创建机器人。

7. 填写机器人名称（如 "OpenClaw 助手"），设置可见范围
8. 权限配置：开启「接收消息」和「发送消息」
9. 点击 **「保存」**

### Step 2：在 OpenClaw 服务器上安装企微官方插件

SSH 登录到 OpenClaw 服务器，执行：

```bash
# 教学需要, 锁定版本, 如需要更新请移除锁定wecom-openclaw-plugin版本 --version 2026.4.23
npx  -y @wecom/wecom-openclaw-cli install  --version 2026.4.23
```

安装过程中按提示输入 Step 1 中获取的 **Bot ID** 和 **Secret**。

**备选安装方式**（如果上面的命令不工作）：

```bash
# 方式 A：通过 OpenClaw 插件命令安装
## 教学需要, 锁定版本, 如需要更新请移除锁定 版本 `@2026.4.23` 标记 -> openclaw plugins install @wecom/wecom-openclaw-plugin
openclaw plugins install @wecom/wecom-openclaw-plugin@2026.4.23

# 然后手动添加通道
openclaw channels add
# 选择 WeCom → 输入 Bot ID → 输入 Secret
```

**方式 B：直接编辑配置文件**

```bash
nano ~/.openclaw/openclaw.json
```

在 `channels` 节点下添加：

```json
"wecom": {
    "enabled": true,
    "botId": "你的Bot ID",
    "secret": "你的Secret"
}
```

### Step 3：重启网关

```bash
systemctl restart openclaw
```

查看日志确认连接成功：

```bash
journalctl -u openclaw -f
```

看到类似以下日志即表示连接成功：

```
[wecom] [default] Authentication successful
```

### Step 4：用户配对（首次使用）

1. 打开企业微信 App
2. 找到刚创建的机器人（工作台 → 智能机器人，或直接搜索机器人名称）
3. 发送第一条消息

机器人可能会回复一个 **配对码（Pairing Code）**。将配对码输入回复即可完成配对。

> 配对码在 OpenClaw Dashboard 的 Settings → DM Pairing 中也可以查看和生成。

### Step 5：群聊配置

1. 在企微中创建或进入一个群聊
2. 群设置 → 添加群成员 → 搜索你的机器人名称 → 添加到群
3. 在群里 **@机器人** 发送消息即可触发回复

> **注意：** 群聊中机器人只会响应被 @的消息，不会监听群里的所有消息。

## 四、常见问题排查

| 症状 | 原因 | 解决方案 |
|------|------|----------|
| 没有 API 模式选项 | 不是管理员 | 确认你是企业微信管理员权限 |
| 连接超时 / Authentication failed | Bot ID 或 Secret 错误 | 重新从企微后台复制凭证 |
| 发消息没回复 | 配对未完成 / 网关未启动 | 检查配对状态，查看日志 |
| 群聊中 @机器人无响应 | 机器人未加入群 / 权限不足 | 确认机器人已添加到群，且开启了消息权限 |
| 其他群友 @机器人无响应 | 可能是 DM 策略限制 | 检查 dm.mode 配置，可设为 open |

## 五、Session 隔离说明

企业微信和微信 ClawBot 共享同一个 OpenClaw Agent，但对话 Session 是隔离的：

- **微信私聊** = 独立 Session A
- **企微私聊** = 独立 Session B
- **企微群聊** = 独立 Session C

三者互不干扰。但 Agent 的人格（SOUL.md）和长期记忆（MEMORY.md）是全局共享的——Agent 在任何入口都"认识你"，只是不记得你在另一个入口的具体对话内容。

## 六、进阶：通过个人微信访问企微机器人

如果你的微信 ClawBot 还没灰度到，可以通过以下路径在个人微信中使用企微机器人：

1. 在企微管理后台 → **我的企业** → **微信插件**
2. 右侧「邀请关注」处会显示企业二维码
3. 用个人微信扫码关注企业
4. 在个人微信中进入企业，找到你创建的机器人
5. 开始对话

> 注意：通过微信插件访问的企微应用只支持私聊，不支持拉入微信群。

---

## 七、安全提醒

- 凭证（Bot ID、Secret）通过环境变量或配置文件管理，不要硬编码在脚本中
- 如果配置文件在 Git 仓库中，确保 `openclaw.json` 已加入 `.gitignore`
- 群聊场景中建议保持 @触发模式，避免 Agent 对每条消息都回复导致刷屏和 Token 浪费
